# 北斗时空信息生态图谱门户 (Beidou EcoLink Portal)

这是一个专为北斗时空信息行业打造的专业生态门户与资源管理平台。

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/FFICAN888/company-ecosystem-directory)

## 🌟 核心特性

- **生态全景视图**：多维度展示企业、高校、甲方及生态伙伴的关联关系。
- **智能资源提报**：标准化的北斗资源提报流程，包含核心能力、技术脉络与协作诉求。
- **管理 review 机制**：内置管理控制台，支持管理员对入库资源进行审核与管理。
- **极简交互设计**：基于 React 19 + Tailwind CSS 4.0 打造的高端、动态视觉体验。

## 🛠 技术栈

- **Frontend**: React 19 + TypeScript + Vite
- **Styling**: Tailwind CSS 4.0 (支持深色模式与毛玻璃特效)
- **Icons**: Lucide React
- **Infrastructure**: Vercel (Recommended Deployment)

## 🚀 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/FFICAN888/company-ecosystem-directory.git
   cd company-ecosystem-directory
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发服务器**
   ```bash
   npm run dev
   ```

## 📂 目录结构

- `src/CompanyEcoSearch.tsx`: 核心业务组件（包含搜索、提报、管理后台）。
- `src/App.tsx`: 应用入口。
- `src/index.css`: Tailwind 基础样式。
